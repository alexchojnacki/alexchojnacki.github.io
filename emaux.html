Nomenclature des tests

Format de l'ID

B3-Cu2-Fe1-C8-03

Chaque ID est composé de plusieurs parties séparées par des tirets :

1. Base (B1-B6)

Identifie la recette de base utilisée :

  • B1 - Soft Green
  • B2 - Blue Stone
  • B3 - Beurre (base de référence)
  • B4 - Noir Mat
  • B5 - Blanc
  • B6 - Vert

2. Additifs

Liste des oxydes ajoutés avec leur pourcentage :

  • Cu - Cuivre (verts, rouges en réduction)
  • Fe - Fer (bruns, rouilles, céladons)
  • Co - Cobalt (bleus intenses)
  • Ru - Rutile (effets nacrés, moirés)
  • Ni - Nickel (gris, bruns)
  • Ti - Titane (opacifiant, blancs)
  • Zr - Zircon (opacifiant)
  • Sn - Étain (opacifiant, blancs)
  • Cr - Chrome (verts, roses avec étain)
  • Li - Lithium (fondant, bleus)
  • Zn - Zinc (cristallisations)

Exemple : Cu2-Fe1 = 2% cuivre + 1% fer

3. Cône de cuisson (C6-C10)

Température cible de cuisson :

  • C6 - 1222°C
  • C7 - 1240°C
  • C8 - 1263°C (standard)
  • C9 - 1280°C
  • C10 - 1305°C

4. Numéro séquentiel

Numéro unique pour différencier les tests ayant la même composition.

Exemple : 01, 02, 03...

Exemples complets

  • B3-C8-01 → Base Beurre, sans additif, cône 8, premier test
  • B3-Cu2-C8-01 → Base Beurre + 2% cuivre, cône 8
  • B1-Cu3-Ru3-Fe1.5-Co0.5-C8-01 → Soft Green complet
  • B5-Ti5-Zr5-C9-02 → Blanc opacifié, cône 9, 2ème test
